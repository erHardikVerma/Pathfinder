React + Vite UI

A lightning-fast frontend built with React and Vite. It utilizes an App Router architecture and custom Tailwind v4 inline theming within global CSS. Features a strict max-width container and role-based routing protecting secure paths.

8 System Roles

SWAP operates on a strictly controlled Role-Based Access Control (RBAC) system. Each user logs in and is securely routed to their specialized view, preventing data leakage across the supply chain.

Access Levels

1. Admin (Full Ops)
2. Kitchen (Production)
3. Franchise (Ordering)
4. Auditor (Checks)
5. Franchise Staff
6. Kitchen Staff
7. Hygiene Monitor
8. Mclean (Special Ops)

18 Lambda Services

A highly decoupled serverless microservices architecture powered by AWS Lambda. This ensures that a spike in one area (like morning ordering) doesn't crash reporting or auth services.

DynamoDB Storage
